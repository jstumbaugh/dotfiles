# Charles' Dot Files

Configuration of my system the way I like it.

## Installation

1. git clone git://github.com/zorab47/dotfiles ~/.dotfiles
2. cd ~/.dotfiles     ; rake install
3. cd ~/.dotfiles/vim ; git module update --init

## Sources
